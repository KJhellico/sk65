     mini-Readme

Если при старте присутствует файл 0:\Misc\5400.txt, он загружается в EEPROM
и используется для работы. Формат файла аналогичен используемому в EEPROM Tool.
Байты в строке IMSI каждой SIM (от 1-го):
  13 - номер сим для автопереключения
  14 - таймер автопереключения, минут (0 - без авто)
  15 - номер SMS-профиля (от 1)
  16 - номер HTTP-профиля (от 0)
Настройки для физической SIM:
  0x331 - номер HTTP-профиля
  0x332 - номер SMS-профиля
  0x333 - таймер автопереключения, минут
  0x334 - номер SIM для переключения
(Это 2-5 байты 52-й строки файла)
HTTP и SMS-профили можно не указывать заранее, а настроить с телефона
(для каждой виртуальной карты).
Название оператора можно писать в кириллице (Win-1251), но она поддерживается
телефоном только в названии субоператора (по крайней мере, у меня).
